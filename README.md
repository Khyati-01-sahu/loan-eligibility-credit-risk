# loan-eligibility-credit-risk

This project analyzes customer financial data to predict loan approval and credit risk.

## Objective
To help financial institutions make data-driven loan approval decisions.

## Approach
- Data cleaning and missing value handling
- Exploratory data analysis
- Feature selection
- Classification models (Logistic Regression, Decision Tree, Random Forest)

## Output
The model identifies high-risk applicants and highlights important financial risk factors.

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib
